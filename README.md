# Фото Отчёты

PWA для создания отчётов о техническом обслуживании.

## Установка

1. Откройте: [https://taffan.github.io/tocam/](https://taffan.github.io/tocam/)
2. На телефоне: меню браузера → "Добавить на главный экран"
3. Приложение установится как нативное

## Возможности

- Создание отчётов МК, ММ, МА
- Фотофиксация оборудования
- Сканирование штрих-кодов КЕ
- Сохранение черновиков офлайн
- Экспорт в ZIP с XLSX отчётом
- Работает без интернета (после первой загрузки)

## Требования

- Браузер Chrome/Safari
- Права на камеру

## Офлайн

После первого запуска с интернетом приложение кэшируется и работает полностью офлайн.

## Экспорт

- **Отправить** — системный диалог шаринга (iOS/Android)
- **Сохранить** — скачать ZIP файл
