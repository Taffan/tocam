## 🔧 ДОПОЛНИТЕЛЬНО (в сессии 23.07.2026)

### 15. EXIF Orientation — коррекция поворота фото
- `getEXIFOrientation()` читает EXIF из JPEG, применяет поворот/отражение
- Чинит перевёрнутые фото с iPhone

### 16. Процессинг фото — resize 1280px max без обрезки
- Удалён center-crop, полный кадр сохраняется
- Step-down до ≤4096px для iOS (обход падения drawImage)
- `resizeToMax(srcCanvas, w, h)` — новая утилита

### 17. ZXing обновлён до v0.23.0
- `BrowserMultiFormatReader.decodeFromVideoElementContinuously` вместо ручного setInterval+drawImage

### 18. ZXing-оверлеи на iOS
- `renderZXingOverlays()` — кликабельные блоки для выбора ШК
- Коды авто-удаляются через 3 сек

### 19. Post-capture decode + ручной ввод
- `tryDecodeFromCapture()` — ZXing.decode из кадра после нажатия «Сделать фото»
- Если не вышло → prompt() для ручного ввода

### 20. BarcodeDetector — динамический запрос форматов
- `BarcodeDetector.getSupportedFormats()` — фильтр только того, что поддерживает браузер
- Чинит iOS Safari (не поддерживает upc_a, codabar, pdf417 и др.)

### 21. Архив — STORE вместо DEFLATE
- `compression: 'STORE'` в `zip.generateAsync()`
- Чинит бесконечное создание архива на iOS

### 22. Safe area на iOS
- `viewport-fit=cover` в `meta viewport`
- `calc(60px + env(safe-area-inset-top))` padding
