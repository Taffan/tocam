# Проблемы — Фото Отчёты

## 🔴 Не решённые

### 1. Thumbnail в галерее не открывает preview

**Описание:** При нажатии на thumbnail внутри галереи (модалка) — ничего не происходит. Ожидаемое поведение: открывается просмотр фото (preview).

**Поведение до исправлений (15.07.2026):**
- Thumbnail внизу страницы → открывал preview напрямую (работало)
- Thumbnail в галерее → не работал (проблема)

**Поведение после исправлений:**
- Thumbnail внизу страницы → открывает preview напрямую ✅
- Thumbnail в галерее → НЕ работает ❌

**Попытки решения:**

1. `addEventListener` на `.gallery-item` в `renderGalleryGrid`
   - Результат: срабатывает, но `galleryPhotoTap` не открывает preview

2. `onclick` напрямую в HTML разметку
   - Результат: onclick срабатывает, но `currentReport` недоступен из onclick (он внутри IIFE)

3. `window.currentReport` и `window.currentSectionIndex` глобальные
   - Результат: определяются как undefined при вызове из onclick

4. `window.__gallerySection` — сохранение section при открытии галереи
   - Результат: не помогло

**Диагностика:**
- onclick срабатывает (подтверждено через alert)
- `currentReport` undefined при вызове из onclick
- `window.__gallerySection` устанавливается в openPhotoGallery, но используется в onclick

**Статус:** Не решена

**Платформа:** ПК (Chrome, file://), Windows

## 🟡 Частично решённые

### 2. Удаление отчётов с главного экрана

**Описание:** После удаления отчёта — список не всегда обновляется корректно.

## ✅ Решённые

### 3. Закрытие галереи

**Описание:** Кнопки "Назад" и "Готово" в галерее открывали preview вместо закрытия.

**Решение:** `closeGallery` теперь закрывает и галерею, и preview.

---

### 4. Превью модалка перехватывает клики

**Описание:** Скрытая preview модалка перехватывала клики на галерее.

**Решение:** Добавлен `display: none` для `.hidden` модалок.

---

### 5. Кнопка "Назад" в галерее

**Описание:** Не было кнопки "Назад" в галерее.

**Решение:** Добавлен header с кнопкой "← Назад" в галерею.

## 🔍 Как тестировать

1. Отчёт → Секция (например РМД)
2. Нажать на тип фото (например "Фото общего вида РМД")
3. Сделать фото
4. Снова нажать на тип фото → открывается галерея
5. Thumbnails в галерее НЕ открывают preview
