# 📁 Дерево отчёта — структура данных и ZIP-архива

## Структура currentReport (IndexedDB)

```
currentReport (объект в IndexedDB: PhotoReportsDB_v6.reports)
│
├── id:             string   ("l9x2m8r4k1...")
├── created:        string   (ISO дата)
├── modified:       string   (ISO дата)
├── status:         string   ("draft" | "completed")
├── completedAt:    string   (ISO дата, только если status="completed")
├── reportName:     string   ("Заподборье")
├── objectType:     string   ("mk" | "mm" | "ma")
├── equipmentCounts: object  ({ kassa: 2, td: 1, uks: 1, mp: 1 })
├── date:           string   ("2026-07-13")
├── technician:     string   ("Иванов И.И.")
├── comment:        string   (опционально)
├── keCodes:        string[] (["1234567890123", "5901234567890"])
│
└── sections:       Section[]
```

## Структура секции

```
Section
│
├── id:          string   ("kassa_1")
├── name:        string   ("Касса 1")
├── prefix:      string   ("Касса")
├── num:         number   (1)
├── type:        string   ("kassa" | "td" | "tsd" | "uks" | "mp"
│                          | "kassa_zona" | "kso" | "vesi" | "prochee")
├── status:      string   ("pending")
│
├── photoTypes:  PhotoType[]
│   │
│   └── PhotoType
│       ├── id:        string   ("obshiy_vid")
│       ├── name:      string   ("Общий Вид")
│       ├── filename:  string   ("Касса 1 Общий Вид.jpg")
│       ├── multi:     boolean  (true — можно несколько фото)
│       ├── maxPhotos: number   (4 — лимит для multi)
│       └── isKE:      boolean  (true — фото инвентарного номера)
│
└── photos:      Photo[]

    └── Photo
        ├── id:          string   (generateId)
        ├── typeId:      string   ("obshiy_vid")
        ├── photoNumber: number   (1, 2, 3... для multi)
        ├── dataUrl:     string   (data:image/jpeg;base64,...)
        └── timestamp:   string   (ISO дата)
```

## Типы фото по оборудованию

### Касса / Кассовая зона (kassa, kassa_zona)
```
Фото (обычные):
  ├── Общий Вид                       [multi: до 4]
  ├── Общий Вид (покупатель)          [multi: до 4]
  ├── Общий Вид Оборудование          [multi: до 4]
  ├── ККТ                             [multi: до 4]
  ├── Фото Чека                       [multi: до 4]
  ├── Маркировка БП                   [multi: до 4]
  ├── Маркировка оборудования         [multi: до 4]
  └── (kassa_zona: доп. фото ККТ + чек ККМ, маркировка ИБП и ПК, и т.д.)

КЕ (инвентарные номера — через модалку камеры):
  ├── КЕ ККТ
  ├── КЕ ИБП
  ├── КЕ ПК
  ├── КЕ Пин Пад
  ├── КЕ Сканер ШК
  └── КЕ Ручной Сканер ШК
```

### ТД (td)
```
Фото:
  ├── Фото ТД (Общий вид)            [multi: до 4]
  └── Фото ТД (Фокус)                [multi: до 4]

КЕ:
  └── Фото ТД КЕ
```

### ТСД (tsd)
```
Фото:
  └── Фото ТСД (Общий вид)           [multi: до 4]

КЕ:
  └── Фото ТСД КЕ
```

### УКС (uks)
```
Фото:
  ├── Фото общего вида УКС (со стороны АММ)         [multi: до 4]
  ├── Фото общего вида УКС (вид сзади)               [multi: до 4]
  ├── Фото общего вида УКС (вид сзади средняя часть) [multi: до 4]
  ├── Фото общего вида УКС (вид сзади нижняя часть)  [multi: до 4]
  ├── Фото индикации на шлюзе                        [multi: до 4]
  ├── Фото обслуженного ПКД                          [multi: до 4]
  ├── Фото обслуженного Серверного ПК                [multi: до 4]
  ├── Фото сетевого оборудования                     [multi: до 4]
  ├── Фото сетевого оборудования для ТД              [multi: до 4]
  ├── Фото сетевого оборудования доступа в интернет  [multi: до 4]
  ├── Фото переключателя                             [multi: до 4]
  ├── Фото маркировки БП в блоках розеток            [multi: до 4]
  └── Фото ключа ЕГАИС                               [multi: до 4]

КЕ:
  ├── Фото КЕ Серверного ПК
  ├── Фото КЕ ИБП Сервер
  ├── Фото КЕ ПКД
  ├── Фото КЕ ИБП ПКД
  ├── Фото КЕ Монитор
  ├── Фото КЕ Голосовой шлюз
  ├── Фото КЕ Роутер Мастер
  ├── Фото КЕ Роутер Резерв
  ├── Фото КЕ Принтер
  └── Фото КЕ Сканер А4
```

### МП (mp)
```
Фото:
  └── Фото Мобильный принтер + печать(Общий вид)    [multi: до 4]

КЕ:
  └── Фото Мобильный принтер КЕ
```

### КСО (kso)
```
Фото:
  ├── Фото общего вида КСО (общий)                      [multi: до 4]
  ├── Фото тумбы КСО (общий вид, монтаж СКС_кабелей)    [multi: макс 4]
  ├── Фото ККТ + чек ККМ                                [multi: до 4]
  ├── Фото маркировки БП                                [multi: до 4]
  ├── Фото монтажа ТБО                                  [multi: до 4]
  └── Фото обслуженной весовой панели                   [multi: до 4]

КЕ:
  ├── Фото КЕ ККТ
  ├── Фото КЕ ИБП
  ├── Фото КЕ КСО
  ├── Фото КЕ ТБО
  └── Фото КЕ весы
```

### Весы (vesi)

Фото:
  └── Фото Весы напольные         [multi: до 4]

КЕ:
  └── Фото КЕ Весы напольные

### Прочее (prochee)

## 📦 ZIP-архив при экспорте

```
{reportName}_{date}.zip
│
├── report.json
├── КЕ.txt
├── {Формат} {Объект} ЧЛ ТО {Формат} ({Техник}) .xlsx
│
├── КЕ/
│   ├── Касса 1 КЕ ИБП.jpg
│   ├── Касса 1 КЕ ПК.jpg
│   └── ...
│
├── 1# Касса 1 Общий Вид.jpg
├── 2# Касса 1 ККТ.jpg
├── 3# Касса 1 ККТ 2.jpg                (если multi, второе фото)
├── 4# Касса 2 Общий Вид.jpg

├── 6# УКС 1 Фото общего вида УКС (со стороны АММ).jpg
└── ...
```

### XLSX — Лист "Отчет по ТО"

```
| Объект               | Заподборье            |
| Формат               | МК                    |
| Дата                 | 2026-07-13            |
| Провел               | Иванов И.И.           |
|                      |                       |
| Конфигурация объекта |                       |
| Касса                | 2                     |
| ТД                   | 1                     |
| УКС                  | 1                     |
|                      |                       |
| №  | Блок    | Наименование работ              | Статус | Комментарий    |
|----|---------|---------------------------------|--------|----------------|
| 1  | Касса 1 | Общий Вид                       | OK     | нет замечаний  |
| 2  | Касса 1 | ККТ                             | OK     | нет замечаний  |
| 3  | Касса 1 | ККТ 2                           | OK     | нет замечаний  |
| 4  | Касса 1 | Маркировка БП                   | OK     | нет замечаний  |
| 5  | Касса 2 | Общий Вид                       | OK     | нет замечаний  |
| 6  | ТД 1    | Фото ТД (Общий вид)             | OK     | нет замечаний  |
| 7  | УКС 1   | Фото общего вида УКС (вид сзади)| OK     | нет замечаний  |
```

Правила формирования XLSX:
- Заголовок: строки 1-4 (Объект, Формат, Дата, Провел)
- Конфигурация: строки 6+ (только оборудование с кол-вом > 0)
- Таблица: `№ | Блок | Наименование работ | Статус | Комментарий`
- Строки группируются по секциям, внутри секции — сортировка по имени типа фото
- Статус всегда "OK", комментарий "нет замечаний"
- Колонки: № (wch:10), Блок (wch:35), Наименование работ (wch:55), Статус (wch:12), Комментарий (wch:25)

### Формат имён файлов в ZIP

| Условие | Формат | Пример |
|---------|--------|--------|
| Тип МК | `{prefix} {num} {filename}.jpg` | `Касса 1 Общий Вид.jpg` |
| Тип ММ/МА | `{prefix} {num} # {filename}.jpg` | `Кассовая зона 1 # Фото ККТ.jpg` |
| Оборудование без номера | `{prefix} {filename}.jpg` | `УКС Фото индикации на шлюзе.jpg` |
| Multi-фото (2+) | `{filename} {N}.jpg` | `ККТ 2.jpg` |
| КЕ-фото | `{filename}` → папка `КЕ/` | `КЕ/Касса 1 КЕ ИБП.jpg` |
| Обычное фото | `{N}# {filename}` → корень | `3# Касса 1 ККТ.jpg` |
