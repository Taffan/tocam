## Навигация (pageHistory)

```
home → config → data → checklist → photos → ke → checklist → complete → home
                        ↑___________|          ↑________|
```

- `header-back` всегда показывает предыдущую страницу
- `checklist` → клик по секции → `photos`
- `photos` → кнопка "Сохранить" → `checklist`
- `checklist` → "Завершить отчёт" → `complete`

## Управление отчётом

```
1. Создание:    config → data → checklist (currentReport создан)
2. Редактирование: клик по черновику → checklist
3. Завершение:  finishReport() → showComplete() → buildZipBlob()
4. Экспорт:     doShareOrDownload() | downloadArchive()
5. Хранение:    IndexedDB — автосохранение через 2с после изменений
```
